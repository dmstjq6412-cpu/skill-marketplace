## 사용 규칙

- **모든 PR**에 알림을 보내려면 비고란을 비워두거나 "모든 PR 알림 수신"으로 표기
- **특정 PR만** 수신하려면 비고란에 조건을 명시 (예: "feature→main PR만 수신")
- 수신자가 여러 명이면 각각 개별 이메일로 발송

---

## 발송 방식

수신자가 여러 명인 경우 각 수신자에게 동일한 내용으로 개별 발송합니다.
diff-mailer 사용 가능 시 HTML 형식, 없으면 텍스트 fallback으로 발송됩니다.
